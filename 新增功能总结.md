# AgentLinker 新增功能总结

## 📦 本次更新

### 1. 二维码配对 📱

**功能描述：**
- 设备启动时自动生成配对二维码
- 主控端扫描二维码即可完成配对
- 支持终端显示、图片生成、手机扫描

**使用方式：**

```bash
# 方式 1：设备端显示二维码
agentlinker show-qr

# 方式 2：主控端扫描二维码
agentlinker --mode controller
[controller]> qr-pair

# 方式 3：生成二维码图片
agentlinker pairing qr-gen \
  --device-id my-server \
  --pairing-key XK9M2P7Q \
  -o qr.png
```

**优势：**
- ✅ 无需手动输入设备 ID 和配对密钥
- ✅ 降低配对错误率
- ✅ 用户体验更好
- ✅ 适合移动端操作

---

### 2. 局域网自动发现 📡

**功能描述：**
- 使用 UDP 广播/组播自动发现设备
- 实时检测设备上下线
- 支持跨平台（Linux/macOS/Windows）

**使用方式：**

```bash
# 方式 1：交互式发现
agentlinker --mode controller
[controller]> discover

# 方式 2：命令行发现
agentlinker pairing discover --timeout 10

# 方式 3：交互式选择配对
agentlinker pairing discover -i
```

**技术细节：**
- 广播端口：UDP 53535
- 广播地址：255.255.255.255
- 组播地址：224.0.0.251
- 发现间隔：2 秒
- 超时时间：10 秒（可配置）

**优势：**
- ✅ 无需知道设备 IP 地址
- ✅ 自动发现局域网设备
- ✅ 实时显示设备状态
- ✅ 一键配对

## 🚀 快速开始

### 场景 1：控制家里的电脑

**在被控端（家里电脑）：**
```bash
# 安装并启动
curl -fsSL https://your-server.com/install.sh | sudo bash
sudo systemctl start agentlinker

# 查看二维码
sudo journalctl -u agentlinker -f | grep -A 10 "配对密钥"
```

**在主控端（你的手机/笔记本）：**
```bash
# 启动控制器
agentlinker --mode controller

# 发现设备
[controller]> discover

# 或扫描二维码
[controller]> qr-pair
```

### 场景 2：批量管理服务器

**发现所有服务器：**
```bash
[controller]> discover 15

# 选择设备配对
选择设备编号：1
```

**批量执行命令：**
```bash
[controller]> exec server-01 df -h
[controller]> exec server-02 top -l 1
```

## 🔧 API 参考

### Python API

```python
from client.utils.discovery import LANDiscovery, QRCodePairing

# 局域网发现
discovery = LANDiscovery()
discovery.on_device_found = lambda d: print(f"发现：{d.device_name}")
discovery.start_listener(timeout=10)

# 二维码生成
qr = QRCodePairing()
qr_content = qr.generate_pairing_qr(
    device_id="my-device",
    pairing_key="XK9M2P7Q",
    device_name="My Server"
)

# 生成二维码图片
qr.generate_qr_image(qr_content, "qr.png")

# 解析二维码
result = qr.parse_pairing_qr(qr_content)
print(f"设备 ID: {result['device_id']}")
print(f"配对密钥：{result['pairing_key']}")
```

### CLI 命令

```bash
# 生成二维码
agentlinker pairing qr-gen --device-id X --pairing-key Y

# 解析二维码
agentlinker pairing qr-scan '{"type":"agentlinker_pair",...}'

# 发现设备
agentlinker pairing discover --timeout 10

# 交互式发现
agentlinker pairing discover -i

# 广播设备信息
agentlinker pairing broadcast --device-id X --pairing-key Y
```

## 🔐 安全提示

1. **配对密钥** - 8 位动态码，1 小时过期
2. **二维码** - 包含敏感信息，勿公开分享
3. **局域网发现** - 仅在可信网络使用
4. **生产环境** - 建议手动配置，关闭自动发现
