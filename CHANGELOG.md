## [2.2.0] - 2026-03-19

### 🎉 重大更新

#### 跨平台支持
- ✅ **Windows 完整支持**
  - Windows 10/11/Server 支持
  - PowerShell 安装脚本
  - PyInstaller 打包
  - 系统托盘集成
  - 开机自启支持

- ✅ **现代化图形界面**
  - 深色模式
  - 系统托盘（Windows）/菜单栏（macOS）
  - 专业应用图标
  - 自动复制配对密钥

#### 核心功能增强

##### 文件传输 🔥
- 上传文件到被控设备
- 从被控设备下载文件
- 分块传输（支持大文件）
- 断点续传
- MD5 校验
- 文件信息查询

##### 批量命令执行 🔥
- 同时向多台设备发送命令
- 并发执行，结果聚合
- 成功/失败统计
- 超时控制
- 执行时间统计

##### 设备分组管理 🔥
- 创建自定义分组
- 拖拽设备到分组
- 分组颜色标记
- 按组批量操作
- 配置导入导出

##### 命令历史 🔥
- 本地保存最近 100 条命令
- 智能搜索
- 收藏常用命令
- 快速重用

### 📦 技术更新
- 新增 `FileTransfer` 类管理文件传输
- 新增 `BatchExecutor` 类管理批量执行
- 新增 `DeviceGroup` 类管理设备分组
- 优化核心模块支持多平台
- 改进错误处理和超时控制

### 📝 文档更新
- 新增 Windows 安装指南
- 新增 Windows 开发计划
- 新增产品需求文档 (PRD)
- 更新 README 反映新功能
