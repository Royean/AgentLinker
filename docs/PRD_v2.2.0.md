## 📋 产品愿景

**让远程控制变得像聊天一样简单**

任何用户，无论技术背景，都能在 3 分钟内完成安装并开始远程控制设备。

---

## 🎯 v2.2.0 目标

### 核心目标
1. **文件传输功能** - 用户可以在设备间传输文件
2. **批量操作** - 同时控制多台设备
3. **命令历史** - 记录和快速重用常用命令
4. **设备分组** - 按组管理设备，提高组织性

### 用户体验目标
- 文件传输像聊天一样简单（拖拽即可）
- 批量操作一键执行
- 命令历史智能搜索
- 设备分组直观管理

## 📦 v2.2.0 功能需求

### 1. 文件传输功能 🔥 (优先级：P0)

#### 用户故事
> 作为用户，我希望能在控制的设备上上传和下载文件，这样我就不用单独使用 SCP 或 FTP 了。

#### 功能描述
- **上传文件** - 从本地上传到被控设备
- **下载文件** - 从被控设备下载到本地
- **拖拽支持** - 直接拖拽文件到窗口
- **进度显示** - 实时显示传输进度
- **断点续传** - 大文件支持断点续传

#### 技术实现
```python
# 新增指令类型
- file.upload: 上传文件
- file.download: 下载文件
- file.transfer.status: 查询传输状态
- file.transfer.cancel: 取消传输

# 支持的文件类型
- 普通文件：直接传输
- 文件夹：压缩后传输
- 大文件：分块传输
```

#### UI 设计
```
┌────────────────────────────────────┐
│  文件传输                          │
├────────────────────────────────────┤
│  📁 /home/user/downloads/          │
│                                    │
│  📄 file1.txt (2.3 MB) ✅          │
│  📄 file2.zip (15.8 MB) 🔄 45%     │
│  📄 file3.log (1.2 MB) ⏳ 等待     │
│                                    │
│  [选择文件] [上传] [下载] [取消]   │
└────────────────────────────────────┘
```

### 2. 批量命令执行 🔥 (优先级：P0)

#### 用户故事
> 作为用户，我希望能同时向多台设备发送命令，这样我就不用一台一台地操作了。

#### 功能描述
- **选择多台设备** - 勾选要控制的设备
- **批量执行** - 一键向所有选中设备发送命令
- **结果聚合** - 所有设备的结果集中显示
- **成功/失败统计** - 清晰显示执行情况

#### 技术实现
```python
# 批量执行 API
async def batch_execute(device_ids: List[str], action: str, params: dict):
    tasks = [execute(device_id, action, params) for device_id in device_ids]
    results = await asyncio.gather(*tasks, return_exceptions=True)
    return aggregate_results(results)
```

```
│  ☑️ 服务器 2 (🟢 在线)             │
│  ☐ 服务器 3 (🔴 离线)              │
│  ☑️ 我的 Mac (🟢 在线)             │
│                                    │
│  命令：[df -h              ]       │

```

### 3. 命令历史记录 (优先级：P1)

#### 用户故事
> 作为用户，我希望能看到之前执行过的命令，这样可以快速重用，不用重新输入。

#### 功能描述
- **本地历史** - 保存最近 100 条命令
- **云端同步** - 跨设备同步历史记录
- **智能搜索** - 支持关键词搜索
- **快捷重用** - 点击即可填充到输入框
- **收藏命令** - 常用命令可以收藏

#### 数据结构
```json
{
  "command_id": "uuid",
  "command": "df -h",
  "device_id": "server-001",
  "timestamp": 1710864000,
  "result_preview": "/dev/sda1 50G...",
  "favorite": false,
  "tags": ["disk", "check"]
}
```


### 4. 设备分组管理 (优先级：P1)

#### 用户故事

- **拖拽分组** - 拖拽设备到分组
- **分组颜色** - 不同分组不同颜色
- **折叠/展开** - 节省空间
- **按组操作** - 右键分组批量操作

## 🔍 竞争分析

### 类似产品
1. **TeamViewer** - 商业软件，功能强大但复杂
2. **AnyDesk** - 快速，但主要是桌面共享
3. **RustDesk** - 开源，但配置复杂
4. **Tailscale SSH** - 安全，但需要组网

### AgentLinker 优势
- ✅ 更简单：一键安装，自动配置
- ✅ 更开放：完全开源，可扩展
- ✅ 更轻量：无多余功能，专注远程控制
- ✅ 更便宜：完全免费
- ✅ 更安全：配对密钥机制，端到端加密（计划中）

## 📝 开发建议
